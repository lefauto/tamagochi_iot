## Como rodar

Em um ambiente virtual, instale o MQTT
```bash
pip install paho-mqtt
```
Instale Node-RED
```bash
npm install -g --unsafe-perm node-red
```
Rode em terminais separados (no node-red, importe o arquivo json do projeto)
```bash
node-red
python script.py
```
